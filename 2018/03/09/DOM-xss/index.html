DOM-XSS的补充

Mar 9, 2018 | 漏洞挖掘

有别于传统类型的XSS漏洞一般出现在服务器端代码中，DOM-BasedXSS受客户端浏览器的脚本代码所影响。

三种主要的节点：

元素节点，文本节点，属性节点

window.name 属性
document.referer 属性
用户在上一个页面单击链接到达当前页面的URL来源，使用不当会引起XSS
location 属性
location是JavaScript管理地址栏的内置对象，使用location.href能管理页面的URL。
